How to Import
  1. Upload your Excel file (.xlsx or .xls)
  2. Enter the exact sheet name containing your cases (e.g. Cases Status Sheet)
  3. Your file must have columns for: Client and Case Details (required)
  4. Optional columns detected automatically: File No, Tax Period, Assignee, Status, Progress, Completion Date, Invoice Status
  5. New clients and assignees will be created automatically if they don't exist
  6. Existing cases are not duplicated — only new rows are added
